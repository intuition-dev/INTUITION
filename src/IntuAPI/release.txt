# API documentaion

1. To generate human readable documentation install npm documentation globally (`-g`):

```
  npm -g i documentation
```
2. There should be `documentation.yml` file.
    Do command from this folder:

```
  documentation build --config documentation.yml IntuAPI.js -f html -o docs
```

It will generate `docs` folder with the human readable documentation:

http://localhost:9081/IntuAPI/docs/

Copy .nojekyll into docs
